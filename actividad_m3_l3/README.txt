1. ¿Qué tipo de condición te resultó más intuitiva?
La condición que me resultó más intuitiva fue la estructura if – elif – else, 
porque permite evaluar varios casos de forma ordenada. Es fácil de leer y entender, 
ya que el programa va revisando cada condición hasta encontrar la que se cumple. 
Además, es útil cuando hay varias posibles respuestas según un valor.

2. ¿En qué caso usarías condicionales anidadas en un programa real?
Las condicionales anidadas se usarían cuando primero se debe verificar una condición general 
y luego evaluar casos más específicos dentro de ella. Por ejemplo, en un sistema de acceso: 
primero se verifica si el usuario está registrado, y si lo está, dentro de esa condición se puede comprobar si 
la contraseña es correcta o si tiene permisos para acceder a cierta función. Esto permite organizar mejor la lógica del programa.
